addDidTickListener

The addDidTickListener method registers a listener function with the Manager that will be called just after the Manager ticks.

addDidTickListener(listener: TickListener): string
TypeDescription
listenerTickListener

The function that will be called just after the Manager ticks.

returnsstring

A unique Id for the listener that can later be used to remove it.

The provided listener is a TickListener function, and will be called with a reference to the Manager.

Example

This example registers a listener that is called just after each Manager tick.

import {createManager} from 'tinytick';

const manager = createManager();
const listenerId = manager.addDidTickListener(() =>
  console.log('Manager did tick'),
);

manager.start();
// ... wait 250ms, long enough for two ticks to occur
// -> 'Manager did tick'
// -> 'Manager did tick'

manager.stop();
manager.delListener(listenerId);
