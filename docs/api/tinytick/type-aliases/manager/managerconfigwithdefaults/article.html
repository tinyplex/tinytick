ManagerConfigWithDefaults

The ManagerConfigWithDefaults type represents the full configuration for the Manager as a whole, including any defaults of those you yourself have not provided.

{tickInterval: DurationMs}
TypeDescription
tickIntervalDurationMs

The time in milliseconds after one tick completes before the next tick will start, defaulting to 100.

It currently has just one property, which lets you indicate the tickInterval. This represents the time between Manager 'ticks' - essentially the heartbeat of how often the Manager should check for new tasks to run, or existing ones to abort.
