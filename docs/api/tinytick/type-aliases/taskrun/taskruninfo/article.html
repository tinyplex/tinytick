TaskRunInfo

The TaskRunInfo type represents information about a scheduled or running task run.

{
  taskId: Id;
  arg: string | undefined;
  retry: number;
  running: boolean;
  nextTimestamp: TimestampMs;
}
TypeDescription
taskIdId
argstring | undefined
retrynumber
runningboolean
nextTimestampTimestampMs

Its properties include the Task Id, any argument provided, whether it is currently running, and so on.
