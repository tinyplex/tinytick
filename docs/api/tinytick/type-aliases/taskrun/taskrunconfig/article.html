TaskRunConfig

The TaskRunConfig type represents a configuration you can provide for a task run.

{
  maxDuration?: DurationMs;
  maxRetries?: number;
  retryDelay?: DurationMs | string;
}
TypeDescription
maxDuration?DurationMs
maxRetries?number
retryDelay?DurationMs | string

This can be provided for a specific task run, a Task as a whole, or for all tasks in a category.

Individual configuration items for a task run will override that of the Task, which in turn override those of a task category, if any. The Manager itself provides defaults if none are provided at any level.

This has properties which let you indicate the duration of task runs and their retry behaviors, for example.
